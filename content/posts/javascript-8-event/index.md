## 1. Event là gì?
Event là hành động của người dùng như click, nhập liệu.

## 2. Các event phổ biến
- onclick
- onchange

## 3. Ứng dụng
Form đăng nhập, nút bấm.

## 4. Kết luận
Event giúp website trở nên tương tác hơn.
