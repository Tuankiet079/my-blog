## 1. Khái niệm OOP 
Lập trình hướng đối tượng (OOP) giúp xây dựng chương trình dễ bảo trì và mở rộng.

## 2. Các đặc điểm chính
- Encapsulation (Đóng gói)
- Inheritance (Kế thừa)
- Polymorphism (Đa hình)
- Abstraction (Trừu tượng)
