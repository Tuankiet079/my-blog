## 1. Java là gì?
Java là một ngôn ngữ lập trình hướng đối tượng, được phát triển bởi Sun Microsystems. Java nổi bật với khả năng chạy trên nhiều nền tảng nhờ JVM (Java Virtual Machine).

## 2. Vì sao Java phù hợp lập trình mạng?
Java cung cấp thư viện mạnh mẽ cho lập trình mạng như `java.net`, hỗ trợ Socket, URL, TCP, UDP.

## 3. Ứng dụng thực tế
- Web Server
- Ứng dụng Client – Server
- Hệ thống phân tán

## 4. Kết luận
Java là lựa chọn phổ biến trong các hệ thống mạng nhờ tính ổn định và bảo mật cao.
